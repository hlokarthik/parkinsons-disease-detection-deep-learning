# Parkinson’s Disease Detection using Deep Learning

This project focuses on detecting Parkinson’s Disease using voice-based biomedical features such as Jitter and Shimmer. The model analyzes voice measurement data to help identify possible Parkinson’s Disease patterns through data analysis and machine learning techniques.

## Features
- Voice feature analysis
- Parkinson’s Disease prediction
- Data preprocessing and visualization
- Exploratory Data Analysis (EDA)
- Biomedical dataset handling

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

## Dataset
The dataset contains biomedical voice measurements including features like:
- Jitter
- Shimmer
- Frequency measurements
- Other vocal attributes related to Parkinson’s Disease detection

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Analysis
5. Model Development
6. Prediction and Evaluation
